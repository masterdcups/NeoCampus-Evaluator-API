HELP WEBPAGE - API NEOCAMPUS EVALUATOR

/!\ All methods are using the POST HTTP request /!\

The main url is '/' and needs the following entry : {"luminosity":your_var, "humidity":your_var, "co2":your_var, "temperature":your_var}

If you want one of the 4 variables you should use one of those url :

  • /luminosity
  • /humidity
  • /co2
  • /temperature
and the following entry according to the url. For example : https://api-neocampus-confort.herokuapp.com/luminosity will need : {"luminosity":your_var}.
